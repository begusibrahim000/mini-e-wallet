### MINI E-WALLET


Frameworks: Express.JS

How to run the App: 

- clone the repo
- copy .env.example to .env
- create mysql db `mini_ewallet`
- run `yarn install`
- run `sequelize db:migrate`
- run `sequelize db:seed:all`
- run `yarn start`
- import postman collection
- try the API endpoint on your postman app
